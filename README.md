# 奶茶店管理系统 (Milk Tea Management System)

一个功能完整的奶茶店管理系统，用于管理订单、库存、员工和销售数据。

## 项目结构

```
milk-tea-management/
├── frontend/          # 前端代码
│   └── README.md     # 前端说明文档
├── backend/           # 后端代码
│   └── README.md     # 后端说明文档
└── docs/             # 项目文档
```

## 功能特性

### 前端功能
- 📱 响应式设计，支持移动端和桌面端
- 🎨 现代化UI界面
- 📊 实时数据展示
- 🛒 订单管理界面
- 📈 销售数据可视化

### 后端功能
- 🔐 用户认证与授权
- 📦 库存管理
- 👥 员工管理
- 💰 财务管理
- 📋 订单处理
- 📊 数据统计分析

## 技术栈

### 前端
- React / Vue.js / Angular
- TypeScript
- Tailwind CSS
- 状态管理
- HTTP客户端

### 后端
- Node.js / Python / Java
- 数据库 (MySQL/PostgreSQL/MongoDB)
- RESTful API
- JWT认证
- ORM/ODM

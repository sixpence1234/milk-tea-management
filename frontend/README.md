# 前端代码目录

这个目录用于存放奶茶店管理系统的前端代码。

## 建议的技术栈
- React/Vue.js/Angular
- TypeScript
- Tailwind CSS / CSS Modules
- 状态管理 (Redux/Vuex/Pinia)
- HTTP客户端 (Axios)

## 目录结构建议
```
frontend/
├── src/
│   ├── components/     # 公共组件
│   ├── pages/         # 页面组件
│   ├── store/         # 状态管理
│   ├── services/      # API服务
│   ├── utils/         # 工具函数
│   ├── assets/        # 静态资源
│   └── styles/        # 样式文件
├── public/            # 公共文件
├── package.json       # 依赖配置
└── README.md         # 项目说明
```

## 功能模块
- 用户登录/注册
- 订单管理
- 库存管理
- 员工管理
- 销售统计
- 设置配置
